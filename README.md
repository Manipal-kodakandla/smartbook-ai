# SmartBook AI 🧠

SmartBook AI is a system that converts raw study materials like PDFs into structured learning content with explanations and examples.

## 🚀 Features
- Upload PDF or text
- Extract content
- Generate topics using AI
- Simplify explanations
- Highlight important keywords

## 🛠 Tech Stack
- Python
- NLP
- Gemini API
- MySQL (if used)

## ⚙️ How it works
1. User uploads document
2. Text is extracted and cleaned
3. AI processes content
4. Topics and explanations are generated

## ⚠️ Challenges I faced
- PDF text extraction issues  
- AI responses were inconsistent  
- Fixed using preprocessing and better prompts  

## ▶️ Run locally
- Clone repo
- Install dependencies
- Add API key
- Run script
